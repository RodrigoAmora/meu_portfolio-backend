# meu-portfolio-backend
Meu site de portfolio criado usando PHP com o framework Laravel 5.6.17 <br>

Instalando as dependências:
-------------------------
`composer update --no-scripts`

Rodando o servidor:
-------------------
`php artisan serve --host=0.0.0.0 --port=4200`
